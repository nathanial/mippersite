This is Mipper, the Online Mips Interpreter: a somewhat buggy implementation of the Mips Assembly Language by Nathan Hartman. The source code for Mipper can be found at github.com/nathanial/mipper while the source code for the website itself can be found at github.com/nathanial/mippersite.

Please note that is this a super mega alpha version, which means that both the interpreter and the website are sure to be chock full of bugs. If you are so adventurous as to try out any of the functionality, please send any bug reports to nathanial.hartman@gmail.com

Further Details: Both the interpreter and this website were written in Python. The website uses Django, and is hosted on Google Appengine. The template for this website was created by willr--as per his request, there is a link to his website at the foot of this page. For the interpreter, the excellent Yapps2 parsing library was used. Special thanks to the Academy, my agent, my producer, all the great members of the cast, and now... the end.

Final Warnings: this site is best viewed using firefox. For whatever reason, it can go a little nuts on Internet Explorer and Apple Safari. Hopefully I'll figure out what the problem is and fix it soon.
